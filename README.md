Project Description
The goal of the project is to build a fictional encyclopedia platform about supernatural powers, where users can create, edit, search and share entries. The fictional encyclopedia is an interest-driven, creative hub for all interested users. In this application, people can work together to create and maintain a shared knowledge database for a fictional world. It is not only an encyclopedia, but also addresses the emotional and cultural needs of current young people.

Application platform
This project will use the Indexes Oracle relational database to store and manage data. Executed stored procedures, triggers.For the application, use HTML, CSS and JAVA Script for the front-end, and PHP for the back-end. Enhanced content search experience integrating ElasticSearch with Inverted Index.
